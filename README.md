# マレー語学習アプリ

Vite + React で構築されたマレー語学習アプリケーションです。

## 機能

- 単語カード表示
- 正式形/日常の短縮形の切り替え
- 例文表示（各単語に2つの例文）
- Web Speech API を使用した音声再生（マレー語）

## セットアップと実行

以下のコマンドを順番に実行してください：

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` を開いてアプリを確認できます。

## ビルド

本番用のビルドを作成する場合：

```bash
npm run build
```

## 使用技術

- Vite
- React
- Tailwind CSS
- Web Speech API
